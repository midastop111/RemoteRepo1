# Git과 GitHub Study를 위한 프로젝트

## 1. 프로젝트 소개
SpringFramework MVC 게시판 기능을 주제로 Git과 GitHub Study를 위한 프로젝트
___

## 2. 개발기간
2023년 04월 17일 ~ 2023년 04월 28일
___

## 3. 멤버구성
* 팀장 왕코딩 : 프로젝트 구조 설계 및 관리
* 팀원 어머나 : 게시판 메인 페이지
* 팀원 한기술 : 게시판 쓰기 및 수정
___

## 4. 개발환경
* Java 8
* JDK 1.8.270
* Tomcat 9
* Framework : SpringFramework 4.2
* Database : MySQL 8
* ORM : MyBatis
___

## 5. 주요기능
* 게시 글 리스트
* 게시 글 쓰기, 수정, 삭제
* 페이징 처리
* 키워드 검색 기능
* 동적쿼리 적용
